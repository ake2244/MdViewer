# MdViewer

*На других языках: [English](README.md)*

Маленький и быстрый просмотрщик Markdown для Windows с редактированием,
печатью и экспортом в PDF. Вся программа занимает меньше 2 МБ и не требует
установки чего-либо: используются .NET Framework 4.8 и движок Edge WebView2,
уже встроенные в Windows 10/11.

![Скриншот MdViewer](docs/screenshot.png)

## Возможности

- **Просмотр** Markdown-файлов с оформлением в стиле GitHub: заголовки,
  списки, таблицы, блоки кода, цитаты, картинки (включая относительные пути)
- **Редактирование** в простом текстовом режиме «как в блокноте» (`Ctrl+E`)
  с защитой от потери несохранённых изменений; исходный стиль концов строк
  (LF/CRLF) сохраняется при записи
- **Печать** на любой принтер с предпросмотром или **экспорт в PDF** в один клик
- Открытие через диалог (`Ctrl+O`), перетаскивание в окно, командную строку
  (`MdViewer.exe файл.md`) или двойной клик (после ассоциации файлов)
- Внешние ссылки открываются в вашем браузере; ссылки на соседние `.md`
  файлы — прямо в просмотрщике
- Русский интерфейс

## Скачать

Готовая сборка — на странице [Releases](../../releases): распакуйте в любую
папку и запустите `MdViewer.exe`. Установка не требуется.

**Требования:** Windows 10/11 x64. .NET Framework 4.8 и среда WebView2
входят в состав Windows 11 (на старых сборках Windows 10 может понадобиться
установить [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)).

## Сборка из исходников

Visual Studio 2022 (с нагрузкой «Разработка классических приложений .NET»)
или .NET SDK:

```
dotnet build MdViewer.sln -c Release
```

Результат появится в `MdViewer/bin/Release/net48/`.

Зависимости (подтягиваются из NuGet автоматически):

- [Markdig](https://github.com/xoofx/markdig) — преобразование Markdown в HTML
- [Microsoft.Web.WebView2](https://developer.microsoft.com/microsoft-edge/webview2/) — отображение
